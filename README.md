# Portfolio Website

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/your-username/portfolio-website.git
    cd portfolio-website
    ```

2. Install the dependencies:
    ```sh
    npm install
    ```

3. Run the project:
    ```sh
    npm start
    ```
